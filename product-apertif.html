The process

Vibe-coded with AI — previewed locally, iterated in conversation, and shipped through version control.

Created in conjunction with Cursor's AI coding tools and Figma's design based AI coding tools, I previewed the websites in local host as I asked the AI to tweak and iterate the site's content, appearance, and behavior.

Process 01

Design.md Guidelines

Using images from the web, text in specific fonts, and hexcodes, I compiled a ‘moodboard’ and developed a tool to have the Cursor AI analyze and extract relevant design features, and then compile that data into a guiding document for future website development. The AI needs only have access to the design.md file, and it will generate or convert any website into a new layout, theme, color, and font family to match the document’s specifications.

Process 02

GitHub Repository

In order to upload and launch code between my computer and the web, I utilized GitHub Repositories to manage version numbers, instantly update websites, and even launch websites using Github's Pages feature.
